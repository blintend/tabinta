Installation

The extension has been written for Firefox browser. Mozilla is not supported currently.

tabinta-0.2.xpi
Mozilla support.
tabinta-0.1.xpi
Firefox 2 compatibility.
tabinta-0.0.7.xpi
Bugfix: version 0.0.6 inserted tab in normal (one-line) input boxes, not just in textareas.
tabinta-0.0.6.xpi
Firefox 1.5 compatible.
Preferences GUI added for tab key configuration.
tabinta-0.0.5.xpi
Added hidden preference to configure the key combination used to insert the tab character.
tabinta-0.0.4.xpi
Tab key behavior can now be toggled by a textarea context menu item.
tabinta-0.0.3.xpi
Added hidden preference to toggle tab insertion behavior.
tabinta-0.0.2.xpi
Initial public version.
