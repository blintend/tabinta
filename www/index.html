What is Tabinta?

Common browsers, including the ones from the Mozilla family, use the tab key to navigate through links and form controls on the page. This behavior, while nice for consistency, prevents the real tab character to be entered in input fields. (Instead of inserting the character, tab key moves focus to the next field.) See Mozilla bug #29086 for details.

Tabinta (which stands for "Tab in Textarea") is a simple Firefox extension which adjusts the tab key behavior; it inserts itself when pressed in a textarea; at other places it moves focus in the usual way.
